# Инструкция по установке Firefox Addon

## Быстрая установка

### Сначала соберите XPI

```bash
./build.sh
```

### Проверка XPI (опционально)

```bash
./validate-xpi.sh
```

### Установка в Firefox

1. Откройте Firefox
2. В адресной строке введите: `about:debugging`
3. В левом меню выберите "This Firefox"
4. Нажмите кнопку "Load Temporary Add-on..."
5. Выберите файл `dist/hrscrape2mart.xpi`

### Если Firefox говорит, что XPI коррумпирован

1. Убедитесь, что вы используете последнюю версию XPI: `./build.sh`
2. Проверьте валидность: `./validate-xpi.sh`
3. Попробуйте пересоздать XPI: `rm dist/hrscrape2mart.xpi && ./build.sh`
4. Убедитесь, что все файлы на месте (manifest.json, все .js файлы, popup.html)
6. Расширение будет установлено временно (до перезапуска Firefox)

## Создание иконок

Для работы расширения нужны иконки. Создайте два файла в папке `icons/`:
- `icon-48.png` (48x48 пикселей)
- `icon-96.png` (96x96 пикселей)

Можно использовать любой графический редактор или онлайн-генератор иконок.

## Использование

1. Откройте страницу Get on Board со списком вакансий (например: https://www.getonbrd.com/myjobs)
2. Нажмите на иконку расширения в панели инструментов Firefox
3. Нажмите "Scrape Job List" - расширение извлечет все ссылки на вакансии
4. Нажмите "Enrich Jobs" - расширение откроет каждую страницу вакансии и извлечет детальную информацию
5. Нажмите "Copy TSV to Clipboard" - результат будет скопирован в буфер обмена
6. Вставьте TSV в Google Sheets (вставьте в ячейку A1, затем Data > Split text to columns)

## Примечания

- Процесс обогащения может занять время, так как открывается каждая страница вакансии
- Все данные сохраняются локально в браузере
- Используйте "Clear Data" для очистки сохраненных данных
